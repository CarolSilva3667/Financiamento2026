# Financiamento2026

## Descrição
O projeto foi feito para poder simular um financiamento, ou seja, ele vai ajudar o usuário
a ver quantas parcelas seram mais vantajosas para ele, qual o valor do bem que será financiado, 
qual seria a taxa de juros para aquele valor e qual será o valor das taxas adicionais.

## Tecnologias
- Vs Code;
- Dart;
- Chrome (web.javascript);
- Figma;

## Como Executar
1. Abra o Vs Code;
2. Procure a pasta "lib";
3. Dentro da pasta LIB clique em "main.dart";
4. Na barra embaixo procure por "No Device" ou "Windows *windows-x64";
5. Ao encontrar altere para "Chrome (web.javascript)";
6. Após alterar inicie o projeto no "Start Debugging" ou Play;
7. Se tiver realizado corretamente o projeto abrirá no chrome;
